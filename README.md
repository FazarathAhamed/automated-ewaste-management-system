# ♻️ Automated E-Waste Management System (AEWMS)

![Python](https://img.shields.io/badge/Python-3.x-blue)
![PEP8](https://img.shields.io/badge/Code%20Style-PEP8-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

A console-based Python application for managing 
electronic waste collection operations.

## ⚡ Features
- 📋 Add / Update / Delete e-waste records
- 💰 Fee calculator with 5% bulk discount (>50kg)
- 🏭 Storage monitoring with 1000kg capacity limit
- ⚠️ Hazard alert for items stored over 30 days
- 📊 Auto-generated summary report (report.txt)
- 🔍 Search by ID or device name

## 🚀 Run
```bash
python aewms.py
```
